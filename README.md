# Web Scraping Application – Book Data Extracting

## Overview
This Python application scrapes product data from an e-commerce website and stores it in a structured format for analysis.


## Features
- Fetches HTML content using HTTP requests
- Parses structured data with BeautifulSoup
- Extracts titles, prices, and ratings
- Stores results in CSV format

## Technologies
- Python
- Requests
- BeautifulSoup
- Pandas

## Project Structure
- `scraper/` – core scraping logic
- `main.py` – application entry point
- `data/` – output files

## How to Run
```bash
python main.py
